jQuery Loading Button Plugin

A simple plugin to add loading state to buttons when clicked. Click the buttons for demos. Check the code below for code examples.

Demo

Example code

$('#btn-1').loadingButton({
	disable: true,
	loadingText: 'Loading...',
	loadingClass: 'btn-loading'
});

$('#btn-2').loadingButton({
	loadingText: 'l0ad1ng...',
	disable: false,
	loadingClass: 'btn-loading'
});

<input type="button" id="btn-3" data-loading-button="true" data-loading-text="LOADING DATA" value="Button 3">

<button id="btn-4" data-loading-button="true" data-loading-text="LOADING DATA!!!">Button 4</button>

Documentation

Simple usage via data attributes. Add attribute data-loading-button to automatically hook up JavaScript click handler. Set the text of the button with data-loading text.

<input type="button" data-loading-button="true" data-loading-text="Loading...">

More advaced use with jQuery plugin method and options.

$('#my-button').loadingButton({
	disable: true, // Set to true or false if the button should be disabled while loading
	loadingText: 'Loading...', // Set the text of the button while loading
	loadingClass: 'loading' // Set the class that should be added to button while loading,
	condition: function() {  } // Function that returns true or false. If false button will not get loading state.
});

Options

Option Default value Description
disable true Set to true or false. Adds disabled attribute to button if true.
loadingText null The text that should be displayed when the button is clicked and gains loading state.
loadingClass null The class that should be added to the button when it is clicked and gains loading state.
condtion true A function that returns true or false. If false the button will not gain the loading state. This can be used if you want to validate a condition before setting the loading state. For example if you want to validate a form.
